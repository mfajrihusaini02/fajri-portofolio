## Vision

Membangun website portfolio pribadi yang ringan, cepat, profesional, dan SEO-friendly untuk menampilkan pengalaman, kemampuan teknis, serta project yang pernah dikerjakan.

Website ini berfungsi sebagai representasi profesional di internet dan memudahkan recruiter, hiring manager, client, maupun rekan developer untuk mengenal profil dan kemampuan saya.

---

## Primary Goal

1. Menampilkan identitas profesional.
2. Menampilkan pengalaman kerja.
3. Menampilkan project terbaik.
4. Menampilkan teknologi yang dikuasai.
5. Menjadi media personal branding.

---

## Non Goal

Website ini bukan:

* Blog
* E-Commerce
* Dashboard
* Admin Panel
* Social Media
* Learning Platform
* CMS

Tidak ada fitur yang tidak berhubungan dengan portfolio.
